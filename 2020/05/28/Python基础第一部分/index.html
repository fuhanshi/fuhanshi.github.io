注意区分map和reduce

reduce是不断地取迭代器中的下一个元素做某个操作，并将函数中前一个值更新为操作后的值，后一个取迭代对象中的下一个值，而map是对每个元素执行一次同样的操作；这里的操作都由那个函数参数指定
